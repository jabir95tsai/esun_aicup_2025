# 模型與特徵工程 (Model)

本資料夾包含核心特徵工程邏輯，負責將清理後的資料轉換為機器學習模型可用的特徵矩陣。

## 檔案說明

| 檔案名稱 | 用途 |
| :--- | :--- |
| `feature_engineering.py` | **特徵工程模組**。<br>主要功能包含：<br>1. **PIT 特徵計算**：基於每個帳戶的觀測點 (Event Date)，計算過去 1/7/30 天內的資金流出/流入統計 (Sum, Count, Mean, Max)。<br>2. **行為穩定性特徵**：計算交易間隔變異係數 (CV)、對手集中度 (HHI)、交易爆發度 (Burstiness)。<br>3. **網絡特徵**：計算帳戶的入度 (In-degree) 與出度 (Out-degree)。<br>4. **相對特徵**：計算個人行為與群體平均的相對比率。 |

## 執行方式
本模組由根目錄的 `main.py` 引用。它依賴 `Preprocess` 模組產生的資料結構。
