# 資料前處理 (Preprocess)

本資料夾包含用於處理原始交易資料、建立觀測點 (Points-in-Time) 以及生成基礎特徵的程式碼。

## 檔案說明

| 檔案名稱 | 用途 |
| :--- | :--- |
| `data_preprocess.py` | **核心前處理模組**。<br>主要功能包含：<br>1. **資料載入**：使用 Polars LazyFrame 高效讀取原始 CSV。<br>2. **觀測點建立**：針對訓練集生成多時間點 (Multi-PIT) 樣本；針對預測集生成單一時間點樣本。<br>3. **基礎清洗**：處理時間欄位、數值型別轉換。<br>4. **二元特徵衍生**：生成如「是否為夜間交易」、「是否為跨行交易」等基礎 Flag 特徵。 |

## 執行方式
本模組通常由根目錄的 `main.py` 自動調用，無需單獨執行。若需進行單元測試，可在根目錄下執行：
```bash
python -m Preprocess.data_preprocess
